# 🌊 수학 놀이에 풍덩! 🌊

초등학교 3학년 2학기 수학을 재미있게 배우는 학습 플랫폼입니다.

## 📚 프로젝트 소개

**수학 놀이에 풍덩!**은 초등학생들이 수학을 흥미롭고 직관적으로 배울 수 있도록 설계된 웹 기반 학습 플랫폼입니다. 게임화된 학습 경험을 통해 아이들이 자연스럽게 수학 개념을 익힐 수 있습니다.

## ✨ 주요 기능

### 현재 완료된 기능
- ✅ **메인 페이지**: 6개 단원을 한눈에 볼 수 있는 대시보드
- ✅ **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기에서 최적화된 화면
- ✅ **단원별 페이지 구조**: 각 단원으로 연결되는 개별 페이지
- ✅ **인터랙티브 UI**: 애니메이션과 호버 효과로 생동감 있는 사용자 경험
- ✅ **키보드 네비게이션**: 숫자 키(1-6)로 빠른 단원 이동 가능

### 페이지 구조
```
index.html              # 메인 대시보드
├── units/
│   ├── unit1.html     # 1단원: 곱셈
│   ├── unit2.html     # 2단원: 나눗셈
│   ├── unit3.html     # 3단원: 원
│   ├── unit4.html     # 4단원: 들이와 무게
│   ├── unit5.html     # 5단원: 분수
│   ├── unit6.html     # 6단원: 그림그래프
│   └── games/
│       ├── unit1/     # 1단원 게임 폴더 (HTML, CSS, JS, 이미지)
│       ├── unit2/     # 2단원 게임 폴더
│       ├── unit3/     # 3단원 게임 폴더
│       ├── unit4/     # 4단원 게임 폴더
│       ├── unit5/     # 5단원 게임 폴더
│       ├── unit6/     # 6단원 게임 폴더
│       └── README.md  # 게임 파일 추가 가이드
├── css/
│   ├── style.css      # 메인 스타일
│   └── unit.css       # 단원 페이지 스타일
└── js/
    └── main.js        # 메인 JavaScript
```

## 📖 단원 구성

### 1단원: 곱셈 ✖️
- 곱셈 퀴즈 게임
- 구구단 마스터
- 곱셈 카드 뒤집기
- 타임 어택 곱셈

### 2단원: 나눗셈 ➗
- 나눗셈 퀴즈 게임
- 똑같이 나누기 게임
- 나눗셈 미로 찾기
- 나눗셈 카드 매칭

### 3단원: 원 ⭕
- 원 그리기 게임
- 중심과 반지름 찾기
- 컴퍼스 시뮬레이터
- 원 모양 찾기 퍼즐

### 4단원: 들이와 무게 ⚖️
- 무게 비교 게임
- 들이 맞추기
- 저울 시뮬레이터
- 단위 변환 퀴즈

### 5단원: 분수 🍕
- 피자 나누기 게임
- 분수 크기 비교
- 분수 퍼즐 맞추기
- 분수 막대 만들기

### 6단원: 그림그래프 📊
- 그래프 만들기 게임
- 데이터 읽기 퀴즈
- 그림 그래프 꾸미기
- 조사하고 그래프 그리기

## 🚀 사용 방법

### 접속 및 시작
1. 웹 브라우저에서 `index.html` 파일을 엽니다
2. 메인 페이지에서 학습하고 싶은 단원을 클릭합니다
3. 각 단원의 놀이를 즐기며 수학을 배웁니다

### 네비게이션
- **마우스/터치**: 단원 카드 또는 "놀이하기" 버튼 클릭
- **키보드**: 숫자 키 1-6으로 해당 단원으로 바로 이동
- **뒤로가기**: 각 단원 페이지의 "← 홈으로" 버튼 또는 ESC 키

## 🎨 디자인 특징

- **컬러풀한 그라데이션**: 각 단원마다 고유한 색상 테마
- **물결 배경**: 역동적인 애니메이션 배경
- **카드 기반 레이아웃**: 직관적이고 현대적인 디자인
- **아이콘과 이모지**: 시각적으로 쉽게 인식할 수 있는 UI
- **부드러운 애니메이션**: 클릭, 호버, 페이지 전환 효과

## 📱 반응형 지원

- **데스크톱**: 1200px 이상 - 그리드 레이아웃으로 모든 단원 표시
- **태블릿**: 768px ~ 1199px - 적응형 그리드
- **모바일**: 767px 이하 - 단일 컬럼 레이아웃

## 🔧 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 플렉스박스, 그리드, 애니메이션
- **JavaScript (ES6+)**: 인터랙션 및 네비게이션
- **Google Fonts**: Noto Sans KR, Gamja Flower

## 🎮 게임 파일 추가 방법

각 단원의 게임은 독립적인 폴더로 구성되어 있습니다:

### 📂 폴더 구조
```
units/games/unit1/
├── index.html      ← 게임 메인 HTML
├── css/            ← 스타일 파일들
│   └── style.css
├── js/             ← JavaScript 파일들
│   └── main.js
└── images/         ← 이미지 파일들
```

### 🚀 추가 방법
1. **게임 폴더 준비**: 완성된 게임의 HTML, CSS, JS, 이미지 파일
2. **파일 복사**: 
   - HTML → `units/games/unit1/index.html`
   - CSS → `units/games/unit1/css/`
   - JS → `units/games/unit1/js/`
   - 이미지 → `units/games/unit1/images/`
3. **경로 수정**: index.html에서 파일 경로 확인
4. **테스트**: 브라우저에서 `units/games/unit1/index.html` 열어서 확인

### 📖 자세한 가이드
- `GAME_SETUP_GUIDE.md`: 상세 추가 방법 및 예시
- `units/games/README.md`: 폴더별 가이드
