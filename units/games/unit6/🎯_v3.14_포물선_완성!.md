
- 💨 **완전 소멸**: scale 0.05, opacity 0
- 🎯 **성공의 쾌감**: 배가됨!

## 📊 주요 변경사항

### 1. 포물선 높이 증가
```javascript
// v3.13
const arcHeight = screenSize * 0.15; // 모든 경우 15%

// v3.14 🆕
const arcHeight = isSuccess ? screenSize * 0.25 : screenSize * 0.15;
// 성공: 25% (67% 증가!)
// 실패: 15% (기존 유지)
```

### 2. 수직 낙하 추가
```javascript
// v3.14 🆕
if (progress > 0.8) {
    // 80~100%: 수직 가속 낙하
    const fallProgress = (progress - 0.8) / 0.2;
    const fallEase = fallProgress * fallProgress; // 가속!
    currentY = arcEnd + (targetY - arcEnd) * fallEase 
               + (fallProgress * 100); // 아래로 빠르게!
}
```

### 3. 빨려들어가는 효과 강화
```javascript
// v3.13
if (progress > 0.7) { // 70%부터, 30% 구간
    scale = 1 - (fallProgress * 0.6); // → 0.4
    opacity = 1 - (fallProgress * 0.7); // → 0.3
}

// v3.14 🆕
if (progress > 0.6) { // 60%부터, 40% 구간
    const shrinkEase = shrinkProgress * shrinkProgress; // 가속!
    scale = 1 - (shrinkEase * 0.95); // → 0.05 (거의 사라짐!)
    opacity = 1 - (shrinkEase * 0.95); // → 0.05 (거의 투명!)
}
```

### 4. 애니메이션 시간 조정
```javascript
// v3.13
const duration = 1000; // 모든 경우 1초

// v3.14 🆕
const duration = isSuccess ? 1200 : 1000;
// 성공: 1.2초 (여유있게 감상)
// 실패: 1.0초 (빠르게 처리)
```
