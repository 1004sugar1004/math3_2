## 🎮 사용 방법

### 그래프 그리기 과정
```
1. 게임 완료 후 결과 화면으로 이동
2. 왼쪽에 큰 콩 주머니 아이콘 1개가 보임
3. 이 아이콘을 첫 번째 모둠의 첫 번째 칸으로 드래그
   → 칸이 채워지고 아이콘은 그대로 남아있음
4. 같은 아이콘을 다시 드래그하여 두 번째 칸에 드롭
   → 또 채워지고 아이콘은 여전히 남아있음
5. 필요한 만큼 반복하여 모든 칸 채우기
6. "완성! 확인하기" 버튼으로 정답 확인
```

### 예시
희망 모둠이 60개를 성공했다면:
- 1칸 = 10개이므로 6칸 채워야 함
- 왼쪽 아이콘을 6번 드래그하여 6칸 채우기
- 완성! ✅

## 🔍 주요 기능 상세

### 1️⃣ 타이밍 게임 시스템
```
콩 주머니 클릭 (1차)
    ↓
파워 게이지 자동 진동 (0% ↔ 100%)
    ↓
적절한 타이밍에 다시 클릭 (2차)
    ↓
파워값에 따라 성공률 결정
    ↓
- 45-65% (초록): 100% 성공률
- 0-44% (빨강): 점점 감소
- 66-100% (빨강): 점점 감소
    ↓
자동으로 던지기 애니메이션
```

### 2️⃣ 단일 재사용 아이콘 시스템
```
[큰 콩 주머니 아이콘 팔레트]
        ↓ 드래그
[그래프 칸에 드롭] → 채워짐
        ↓
[아이콘은 팔레트에 그대로 유지]
        ↓ 다시 드래그
[다른 칸에 드롭] → 또 채워짐
        ↓
무한 반복 가능!
```

### 3️⃣ 정답 확인 시스템
```
각 모둠별로:
채운 칸 수 × 10 vs 실제 점수 비교

예시:
희망 모둠: 6칸 × 10 = 60개
실제 점수: 60개
→ ✅ 정확해요!

미래 모둠: 5칸 × 10 = 50개
실제 점수: 40개
→ ❌ 다시 확인하세요 (10개 더 필요해요)
```
