### 1️⃣ 콩주머니 수 칸이 화면을 꽉 채움!

```
┌────────┬──┬──┬──┬──┬──┬──┬──┬──┬──┬──┐

└────────┴──┴──┴──┴──┴──┴──┴──┴──┴──┴──┘
           10칸이 왼쪽에만 있고 오른쪽 빔
```

**현재 해결:**
```
┌────────┬────┬────┬────┬────┬────┬────┬────┬────┬────┬────┐

└────────┴────┴────┴────┴────┴────┴────┴────┴────┴────┴────┘
           10칸이 화면을 꽉 채움! ✅
```

### 2️⃣ 모둠 열 너비 축소
**이전:**
- 모둠 열: 200px (너무 넓음)
- 콩주머니 칸이 작아 보임

**현재:**
- 모둠 열: **150px** (적절!)
- 콩주머니 칸이 넓어 보임

## 🔧 기술적 수정사항

### HTML (index.html)
```html
<!-- 이전 -->
<th class="header-count" colspan="20">콩 주머니 수</th>

<!-- 현재 -->
<th class="header-count" colspan="10">콩 주머니 수</th>
```
- colspan을 20 → **10**으로 수정
- 헤더가 정확히 10칸만 차지

### CSS (style.css)

#### 1. 모둠 열 축소
```css
/* 이전 */
.graph-table .header-team {
    width: 200px;
}
.team-name-cell {
    width: 200px;
}

/* 현재 */
.graph-table .header-team {
    width: 150px; /* 200px → 150px */
}
.team-name-cell {
    width: 150px;
    min-width: 150px;
    max-width: 150px; /* 고정 크기 */
}
```

#### 2. 테이블 레이아웃 변경
```css
/* 이전 */
.graph-table {
    table-layout: fixed; /* 고정 레이아웃 */
}

/* 현재 */
.graph-table {
    table-layout: auto; /* 자동 레이아웃 */
}
```
- 콩주머니 칸이 남은 공간을 자동으로 채움

#### 3. 그래프 셀 자동 확장
```css
/* 이전 */
.graph-cell {
    width: 60px; /* 고정 크기 */
}

/* 현재 */
.graph-cell {
    width: auto;
    min-width: 50px; /* 최소 크기만 지정 */
}
```
- 남은 공간을 10칸이 균등하게 나눠 가짐

#### 4. 눈금 라벨 정렬
```css
/* 이전 */
.scale-labels {
    padding-left: 200px;
}

/* 현재 */
.scale-labels {
    padding-left: 150px;
    width: calc(100% - 150px); /* 남은 공간 꽉 채움 */
}
```
- 모둠 열 너비에 맞춰 정렬
- 남은 공간을 10칸으로 균등 분할
