## ✨ 핵심 개선사항

### 1️⃣ 자동 순차 채우기 (앞에서부터!)
**이전 방식:**
- ❌ 드래그한 칸에 정확히 놓아야 함
- ❌ 빈 칸 건너뛸 수 있음
- ❌ 순서가 뒤죽박죽

- ✅ 빈 칸 없이 깔끔하게 정렬

```

드래그 2번: 🎒 🎒 _ _ _ (두 번째 칸 채워짐)
드래그 3번: 🎒 🎒 🎒 _ _ (세 번째 칸 채워짐)
...
드래그 12번: 🎒 🎒 🎒 🎒 🎒 🎒 🎒 🎒 🎒 🎒 🎒 🎒 (12개 완성!)
```

## 🔧 기술적 변경사항

### JavaScript (game.js)

#### 1. 자동 순차 채우기
```javascript
function onCellDrop(e) {
    const teamIndex = parseInt(this.dataset.teamIndex);
    
    // 해당 모둠의 모든 셀 가져오기
    const rows = elements.graphTableBody.querySelectorAll('tr');
    const row = rows[teamIndex];
    const cells = row.querySelectorAll('.graph-cell');
    
    // 앞에서부터 빈 칸 찾아서 채우기
    for (let i = 0; i < cells.length; i++) {
        if (!cells[i].classList.contains('filled')) {
            fillCell(cells[i], teamIndex, i);
            break; // 한 칸만 채우고 종료
        }
    }
}
```

#### 2. 클릭 삭제 시 자동 정렬
```javascript
function onCellClick(e) {
    if (this.classList.contains('filled')) {
        const teamIndex = parseInt(this.dataset.teamIndex);
        const cellIndex = parseInt(this.dataset.cellIndex);
        
        // 클릭한 위치부터 뒤의 모든 아이콘을 앞으로 당기기
        for (let i = cellIndex; i < cells.length - 1; i++) {
            if (cells[i + 1].classList.contains('filled')) {
                cells[i].innerHTML = cells[i + 1].innerHTML;
                cells[i].classList.add('filled');
            } else {
                clearCell(cells[i]);
                break;
            }
        }
    }
}
```

#### 3. 단순화된 셀 채우기
```javascript
function fillCell(cell, teamIndex, cellIndex) {
    cell.classList.add('filled');
    cell.innerHTML = `
        <div class="cell-icon">
            <i class="fas fa-shopping-bag"></i>
        </div>
    `;
}
// X 버튼 제거 - 클릭으로 바로 삭제
```
