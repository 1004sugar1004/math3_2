## 🔧 코드 구현

### CSS (3D 훌라후프)
```css
.hulahoop {
    transform-style: preserve-3d;
    transform: rotateX(65deg) rotateZ(0deg);
    animation: rotate3d 10s linear infinite;
}

.hulahoop::before {
    /* 외부 링 */
    border: 15px solid #FFD700;
    box-shadow: 
        0 0 20px rgba(255, 215, 0, 0.6),
        inset 0 -10px 30px rgba(255, 215, 0, 0.4),
        0 15px 40px rgba(255, 215, 0, 0.5);
}

.hulahoop::after {
    /* 내부 그림자 */
    background: radial-gradient(
        rgba(0, 0, 0, 0.3) 0%,
        transparent 70%
    );
}

.hoop-center {
    animation: pulse 2s ease-in-out infinite;
}
```

### JavaScript (빨려들어가는 효과)
```javascript
// 성공 시 3D 효과
if (isSuccess && progress > 0.7) {
    const fallProgress = (progress - 0.7) / 0.3;
    scale = 1 - (fallProgress * 0.6);     // 1.0 → 0.4
    opacity = 1 - (fallProgress * 0.7);   // 1.0 → 0.3
}

beanbag.style.transform = `
    translate(-50%, -50%) 
    rotate(${progress * 720}deg) 
    scale(${scale})
`;
beanbag.style.opacity = opacity;

// 마지막에 완전 소멸
if (progress >= 1 && isSuccess) {
    beanbag.style.opacity = '0';
    beanbag.style.transform = 'scale(0.2)';
}
```

## 💡 기술 하이라이트

### CSS 3D Transform
- `transform-style: preserve-3d`
- `rotateX()` + `rotateZ()`
- Pseudo-elements (::before, ::after)

### JavaScript 애니메이션
- `requestAnimationFrame` (60fps)
- Ease-out cubic curve
- Scale + Opacity 동시 제어

### 성능 최적화
- GPU 가속 (transform)
- 최소 reflow
- 효율적인 애니메이션 루프
