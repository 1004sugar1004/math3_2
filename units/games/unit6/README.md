## 🎯 게임 소개

### 게임 방식 (v3.13 확장)
- **모둠**: 2 ~ 8개 모둠 (사용자 설정)
- **인원**: 각 모둠당 1 ~ 6명 (사용자 설정)
- **던지기**: 한 명당 **1 ~ 30개** (v3.13 확장!) 🆕
- **라운드**: **1라운드 고정** (간단하고 명확)
- **난이도**: 쉬움 / 보통 / 어려움 / 전문가 (사용자 선택)
- **목표**: 훌라후프 안에 최대한 많은 콩 주머니 넣기

### 조작 방법 (타이밍 게임!)
1. **1번 클릭**: 콩 주머니를 클릭하여 파워 게이지 시작
   - 파워 게이지가 자동으로 0% ↔ 100% 왔다갔다
   - 빨강 → 노랑 → 초록 → 노랑 → 빨강 순환
2. **2번 클릭**: 적절한 타이밍에 다시 클릭하여 파워 고정
   - 🟢 **초록 구간 (45~65%)**: 최적! 성공률 최대
   - 🟡 **노랑 구간 (20~45%, 65~85%)**: 보통
   - 🔴 **빨강 구간 (0~20%, 85~100%)**: 위험! 성공률 낮음
3. **자동 던지기**: 파워가 고정되면 자동으로 던져짐
4. **결과**: 파워 + 난이도에 따라 성공/실패 결정!

## ⏱️ 타이밍 게임 시스템

### 🎮 파워 게이지 작동 방식
```
클릭 1: 시작!
↓
파워 게이지 움직임:
0% ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 100%
🔴   🟡    🟢 최적존 🟢    🟡   🔴
     ↑     ↑         ↑     ↑
    20%   45%       65%   85%

클릭 2: 정지!
→ 파워 고정 → 자동 던지기
```

### 🎯 파워 존별 성공률
| 파워 구간 | 색상 | 성공률 보정 | 설명 |
|-----------|------|------------|------|
| 45~65% | 🟢 초록 | 100% | 완벽한 파워! |
| 20~45% | 🟡 노랑 | 50~100% | 약간 약함 |
| 65~85% | 🟡 노랑 | 50~100% | 약간 강함 |
| 0~20% | 🔴 빨강 | 0~50% | 너무 약함! |
| 85~100% | 🔴 빨강 | 0~50% | 너무 강함! |

### 📊 최종 성공률 계산
```
최종 성공률 = 난이도 성공률 × 파워 성공률

예시 1: 보통 난이도(60%) + 최적 파워(100%)
       = 60% × 100% = 60% 성공률

예시 2: 보통 난이도(60%) + 빨강 구간(30%)
       = 60% × 30% = 18% 성공률
       
→ 타이밍이 중요합니다!
```

## 🎯 난이도 시스템

게임의 재미를 위해 **확률 기반** 4단계 난이도를 제공합니다!

|--------|--------|------|-----------|

### 📊 작동 원리
- **확률 기반**: 각 던지기마다 난이도에 따른 확률로 성공/실패 결정

  - ✅ **성공**: 포물선을 그리며 훌라후프 **안쪽**으로 정확히 떨어짐
  - ❌ **실패**: 포물선을 그리며 훌라후프 **바깥쪽**으로 떨어짐
- **랜덤 위치**: 성공/실패 위치가 매번 달라져 더 현실적

3. **"게임 설정"** 버튼을 클릭합니다

### 2단계: 게임 설정
1. **모둠 수 설정**: 2~8개 중 선택 (기본: 4개)
2. **인원 설정**: 각 모둠당 1~6명 (기본: 3명)
3. **던지기 횟수**: 한 명당 **1~30개** (기본: 5개) ← v3.13 확장! 🆕
4. **라운드 수**: **1라운드 고정** (설정 불필요)
5. **난이도 선택**: 쉬움/보통/어려움/전문가 (기본: 보통)
6. **모둠 이름 변경**: 원하는 이름으로 변경 (선택사항)
7. **"게임 시작"** 버튼 클릭

### 3단계: 게임 진행 (타이밍 게임!)
1. **현재 모둠 확인**: 화면 상단에 현재 플레이 중인 모둠 표시
2. **콩 주머니 던지기 (2번 클릭!)**:
   - **1번 클릭**: 콩 주머니를 클릭하면 파워 게이지 시작
   - 파워 게이지가 빠르게 0% ↔ 100% 왔다갔다
   - 🟢 초록 구간(45~65%)을 노리세요!
   - **2번 클릭**: 적절한 타이밍에 다시 클릭
   - 파워가 멈추고 자동으로 던져짐
   - 🎵 던지기 소리가 납니다
3. **결과 확인**:
   - ✅ 성공하면 "성공! 🎉" 메시지와 함께 점수 획득 + 성공 효과음
   - ❌ 실패하면 "아쉬워요! 😢" 메시지 + 실패 효과음
   - 파워가 최적 존에 가까울수록 성공 확률 높음!
4. **턴 진행**:
   - 설정한 개수만큼 콩 주머니 던지기 (1~30개) 🆕
   - 설정한 인원수만큼 플레이어가 차례대로 진행
   - **라운드**: 1라운드 고정
   - **그래프**: 3행 고정 (30칸) (v3.13) 🆕
5. **점수 확인**: 하단 점수판에서 실시간 점수 확인

### 4단계: 결과 확인 및 그래프 그리기 🎨 (v3.13 확장!)
1. **순위 테이블**: 모둠별 순위와 점수 확인 (금/은/동 메달)
2. **3행 30칸 표 형식 그림그래프 그리기**: 🆕
   - **방법 1: 드래그하기**
     - 왼쪽 큰 콩 주머니 아이콘을 **아무 칸에나** 드래그
     - 자동으로 **1행 → 2행 → 3행 순차적으로** 채워짐! 🎯
     - 하나의 아이콘을 여러 번 재사용 가능
     - **1개 아이콘 = 콩주머니 1개** (1:1 대응)
   - **방법 2: 직접 입력하기**
     - 각 모둠 이름 아래 입력창에 숫자 입력 (0~30) 🆕
     - **숫자가 명확하게 보임** ✅
     - **▲▼ 스피너 버튼** 으로도 조절 가능
     - "✓ 적용" 버튼 클릭 또는 엔터키
     - 자동으로 3행에 걸쳐 아이콘 채워짐
   - 잘못 채운 칸 클릭 시 해당 칸부터 **자동으로 앞으로 이동**
   - 지우기 버튼으로 전체 초기화
3. **정답 확인**:
   - "완성! 확인하기" 버튼 클릭
   - 내가 채운 개수 vs 실제 개수 비교
   - 정확도 피드백 (✅ 정확해요 / ❌ 다시 확인하세요)
4. **통계 정보**: 평균, 최고, 최저 점수 확인
5. **다시하기**: 처음 화면으로 돌아가 다시 도전

## 💡 핵심 개선사항 (v2.0 Final)

### 🎯 정확한 시각적 피드백
**이전 문제점**: 
- 던진 콩 주머니가 훌라후프 밖으로 떨어지는데 성공으로 판정되는 등 비직관적
- 성공/실패가 애니메이션과 일치하지 않아 혼란스러움

**현재 해결책**:
- ✅ 먼저 난이도별 확률로 성공/실패를 결정
- ✅ 성공이면 훌라후프 **안쪽**(중심에서 반경 70% 이내)으로 떨어지는 애니메이션
- ✅ 실패면 훌라후프 **바깥쪽**(반경 + 20~80px 떨어진 곳)으로 떨어지는 애니메이션
- ✅ 포물선 궤적을 따라 정확히 목표 지점에 도달
- ✅ **보이는 것이 곧 결과** - 직관적이고 명확함!

### 🎲 확률 기반 난이도
**이전 문제점**:
- 모든 던지기가 거의 성공 → 예측 가능하고 지루함
- 난이도 차이가 실제로 느껴지지 않음

**현재 해결책**:
- ✅ 각 던지기마다 난이도에 따른 확률 적용
- ✅ 쉬움(80%), 보통(60%), 어려움(40%), 전문가(20%)
- ✅ 실제로 성공과 실패가 섞여 긴장감 있는 게임
- ✅ 난이도별 체감 차이가 명확함

## 🎵 효과음 시스템

### Web Audio API 사용
외부 파일 없이 JavaScript로 실시간 효과음을 생성합니다!

#### 던지기 소리 (playThrowSound)
- 주파수: 200Hz
- 타입: 사인파 (sine)
- 길이: 0.3초
- 효과: 휘익~ 소리

#### 성공 소리 (playSuccessSound)
- 3음계: C-E-G (523.25Hz, 659.25Hz, 783.99Hz)
- 타입: 사인파
- 길이: 각 0.3초
- 효과: 따-란! 상승 멜로디

#### 실패 소리 (playFailSound)
- 주파수: 400Hz → 200Hz (하강)
- 타입: 톱니파 (sawtooth)
- 길이: 0.5초
- 효과: 아쉬운 하강 소리

## 🐛 알려진 이슈 및 해결 방법

### AudioContext 경고
**현상**: 콘솔에 "AudioContext was not allowed to start" 경고
**원인**: 브라우저 보안 정책 (사용자 제스처 없이 소리 재생 불가)
**해결**: 정상 작동하며, 첫 클릭 후 효과음이 재생됩니다.

### 모바일에서 드래그 문제
**현상**: 일부 모바일 기기에서 드래그가 부드럽지 않음
**해결**: 터치 이벤트를 별도로 처리하여 대부분 해결되었습니다.
