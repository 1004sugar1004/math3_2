## 🎮 게임 진행 방식

### 간단한 1라운드 게임
```
1단계: 설정
- 모둠 4개
- 인원 3명
- 던지기 5개
- 난이도 보통

2단계: 게임 (1라운드만!)
- 희망 모둠 3명이 각각 5번씩 던지기
- 미래 모둠 3명이 각각 5번씩 던지기
- 최강 모둠 3명이 각각 5번씩 던지기
- 사랑 모둠 3명이 각각 5번씩 던지기
→ 게임 끝!

3단계: 결과 & 그래프
- 각 모둠 점수 확인
- 그래프 작성
- 정답 확인
→ 완성!
```

### 최대 점수
```
최대 인원: 6명
최대 던지기: 10개
라운드: 1회 (고정)

최대 점수 = 6 × 10 × 1 = 60개
→ 10칸 초과하지만 입력 가능
→ 그래프는 10개까지만 표시
```

## 🎯 왜 1라운드?

### 1. 초등학생에게 적합
- ✅ 한 번의 게임으로 결과 확인
- ✅ 기다리는 시간 최소화
- ✅ 집중력 유지 (짧고 재미있게)

### 2. 그림그래프 학습 목적
- ✅ 데이터 수집 → 그래프 작성 → 분석
- ✅ 여러 라운드 필요 없음
- ✅ 한 번의 데이터로 충분

### 3. 수업 시간 활용
```
10분 게임 1회:
- 설정: 1분
- 게임: 5분
- 그래프: 3분
- 분석: 1분
→ 총 10분!
```
