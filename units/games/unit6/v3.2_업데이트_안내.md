## 🎨 개선된 UI

### 1. 모둠 행 축소 + 콩주머니 수 행 확대
```
이전:
┌──────────┬────┬────┬────┐
│          │ 10 │ 20 │ 30 │  ← 작은 칸들
│ 희망모둠 │ 🎒 │ 🎒 │ 🎒 │
└──────────┴────┴────┴────┘
   ↑ 넓은 모둠 행

현재:
┌─────┬──┬──┬──┬──┬──┬──┬──┬──┬──┬──┬──┬──┐
│희망 │🎒│🎒│🎒│🎒│🎒│🎒│🎒│🎒│🎒│🎒│🎒│🎒│ ← 큰 칸들 (1칸 = 1개)
└─────┴──┴──┴──┴──┴──┴──┴──┴──┴──┴──┴──┴──┘
  ↑ 좁은 모둠 행
```

### 2. 크기 조정
- **모둠 이름 셀**: 150px → **120px** (좁게)
- **그래프 셀**: 60px × 70px → **50px × 100px** (넓고 길게)
- **표 행 높이**: 70px → **100px** (더 크게)

## 🎯 새로운 기능: 직접 입력

### 사용 방법
```
1. 각 모둠 이름 아래 입력창 확인
   ┌─────────────┐
   │   희망      │
   │ [12] ✓ 적용 │  ← 여기!
   └─────────────┘

2. 개수 입력 (예: 12)

3. "✓ 적용" 버튼 클릭 또는 엔터키

4. 자동으로 12개 아이콘 채워짐! ⚡
```

### 장점
- ✅ **빠름**: 많은 개수도 한 번에 입력
- ✅ **정확함**: 실수 없이 정확한 개수
- ✅ **편리함**: 드래그하기 귀찮을 때
