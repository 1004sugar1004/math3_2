## 🚀 5가지 핵심 개선

### 1. 포물선 높이 40% 증가
```
v3.14: 25% → v3.15: 35%

           🎒  ← 35% 높이!
          /  \    (기존보다 40% 더 높음)
         /    \
        /      \
       /        \
```

### 2. 수직 낙하 구간 25%로 확대
```
v3.14: 80~100% (20%)
v3.15: 75~100% (25%)

                ⬇️  ← 25% 구간
                ⬇️    (5% 더 김)
                ⬇️    2.5제곱 가속!
```

### 3. 완전 소멸 (scale 0)
```
v3.14: scale 0.05 → v3.15: scale 0

1.0 → 0.4 → 0 (완전히 사라짐!)
      ↑      ↑
   점진적  급격한
   축소    소멸
```

### 4. 애니메이션 시간 1.5초
```
v3.14: 1.2초 → v3.15: 1.5초

25% 증가로 더 여유있게 감상!
```

### 5. 정중앙 타겟팅
```
v3.14: 0~25% → v3.15: 0~10%

완벽한 타이밍 → 정중앙! 🎯
```

- **가속 낙하**: 2.5제곱으로 점점 빠르게

- **쏙! 느낌**: "완전히 빨려들어간다!"

### 4단계: 완전 소멸 (50~100%)
- **점진적 축소**: 1.0 → 0.4 (50~75%)

- **완전 사라짐**: scale 0, opacity 0

## 💪 왜 극적인가?

### 1. 높이의 극대화
- 35% 높이 = 화면 1/3 이상
- 시야에서 확실히 보이는 높이
- 성공의 느낌이 시각적으로 명확

### 2. 낙하의 극대화
- 25% 구간 = 충분한 빨려들어가는 시간
- 2.5제곱 가속 = 점점 더 빠르게
- 150px 추가 = 훨씬 아래로

### 3. 소멸의 극대화
- 50% 구간 = 충분한 축소 시간
- 2단계 소멸 = 점진 + 급격
- scale 0 = 완전히 사라짐

### 4. 시간의 극대화
- 1.5초 = 모든 효과를 여유있게 감상
- 너무 빠르지도, 느리지도 않음
- 완벽한 타이밍

### 5. 정확도의 극대화
- 0~10% = 정중앙!
- 타이밍에 따른 차이가 명확
- 성공의 쾌감 증폭

## 📝 수정 요약

### js/game.js - 5가지 변경

```javascript
// 1. 포물선 높이
screenSize * 0.35 // (v3.14: 0.25)

// 2. 애니메이션 시간
1500 // (v3.14: 1200)

// 3. 수직 낙하
if (progress > 0.5) {
    if (progress <= 0.75) {
        // 포물선 (25%)
    } else {
        // 수직 낙하 (25%)
        Math.pow(fallProgress, 2.5) // (v3.14: 2.0)
        currentY = ... + 150 // (v3.14: 100)
    }
}

// 4. 소멸 효과
if (progress > 0.5) { // (v3.14: 0.6)
    if (progress > 0.75) {
        // 급격한 소멸 (3제곱)
        scale = 0.4 * (1 - finalEase) // → 0
    }
}

// 5. 중심 타겟
0.75 - (accuracyRatio * 0.65) // 0.10 ~ 0.75
// (v3.14: 0.85 - (accuracyRatio * 0.6)) // 0.25 ~ 0.85
```
