## 🎮 핵심 기능 요약

### 1. 완전한 커스터마이징
- 모둠 수: 2-8개
- 모둠당 인원: 1-6명
- 라운드 수: 1-10라운드
- 던지기 횟수: 1-5회
- 난이도: 4단계 (쉬움/보통/어려움/전문가)

### 2. 타이밍 게임 시스템 (V3.1)
- 자동 진동 파워 게이지 (0% ↔ 100%, 60fps)
- 2클릭 메커니즘 (시작/정지)
- 최적 구간 (45-65%): 100% 성공률
- 색상 피드백 (빨강-노랑-초록)

### 3. 확률 기반 던지기
- 파워값 + 난이도 = 최종 성공률
- 성공 → 훌라후프 내부 착지
- 실패 → 훌라후프 외부 착지
- 포물선 애니메이션

### 4. Web Audio API 사운드
- 던지기 사운드 (스윕)
- 성공 사운드 (C-E-G 코드)
- 실패 사운드 (하강 톤)

### 5. 단일 재사용 아이콘 (V3.1 Final)
- 1개 대형 아이콘 (120px)
- 무한 재사용 가능
- 바운스 애니메이션
- "드래그하세요" 라벨

### 6. 표 형식 그림그래프
- 가로 행 (모둠별)
- 1칸 = 10개
- 드래그 앤 드롭으로 채우기
- 클릭으로 삭제

### 7. 정답 확인 시스템
- 각 모둠별 정확도 체크
- 즉각적인 피드백
- 힌트 제공

## 💻 기술 정보

### 사용된 기술
- HTML5 (시맨틱 마크업)
- CSS3 (그라디언트, 애니메이션, 변환)
- JavaScript ES6+ (화살표 함수, 템플릿 리터럴, 구조분해)
- Drag and Drop API
- Web Audio API
- Tailwind CSS (CDN)
- Font Awesome 6 (CDN)
- Google Fonts (CDN)

### 브라우저 요구사항
- Chrome/Edge: 최신 버전
- Firefox: 최신 버전
- Safari: 최신 버전
- 인터넷 익스플로러: 지원 안 함 (ES6+ 사용)

### 성능
- 페이지 로드: ~8초 (CDN 포함)
- 애니메이션: 60fps
- 메모리: ~5MB
- 반응형: 모바일/태블릿/데스크탑
