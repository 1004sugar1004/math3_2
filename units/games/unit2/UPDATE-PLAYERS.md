## 🎮 게임 모드

### 1️⃣ 혼자 플레이
- **대상**: 나눗셈 학습을 원하는 1명
- **특징**:
  - 순서 결정 화면 건너뛰기 (바로 게임 시작)
  - 플레이어 이름: "나"
  - 1번 칸부터 26번 칸(도착)까지 진행
  - 나눗셈 연습 및 학습 목적
  - 승패 없이 도착 지점까지 완주

### 2️⃣ 2명 플레이
- **대상**: 친구, 가족과 함께 대결하고 싶은 경우
- **특징**:
  - 가위바위보로 순서 결정 (자동)
  - 플레이어 이름: "플레이어 1", "플레이어 2"
  - 빨강 vs 청록 색상으로 구분
  - 먼저 도착 칸에 도달한 사람이 승리
  - 경쟁 및 전략적 요소

## 🔧 수정된 코드

### 1. index.html
```html
<!-- Before -->
<button class="player-btn" onclick="selectPlayers(2)">2명</button>
<button class="player-btn" onclick="selectPlayers(3)">3명</button>
<button class="player-btn" onclick="selectPlayers(4)">4명</button>

<!-- After -->
<button class="player-btn" onclick="selectPlayers(1)">혼자 플레이</button>
<button class="player-btn" onclick="selectPlayers(2)">2명</button>
```

### 2. js/game.js - selectPlayers() 함수
```javascript
// 플레이어 수 선택
function selectPlayers(num) {
    playClickSound();
    
    gameState.numPlayers = num;
    gameState.players = [];
    
    for (let i = 0; i < num; i++) {
        gameState.players.push({
            id: i,
            name: num === 1 ? '나' : `플레이어 ${i + 1}`, // ✅ 혼자면 "나"
            color: playerColors[i],
            position: 1,
            order: i + 1 // ✅ 순서 미리 할당
        });
    }
    
    // ✅ 혼자 플레이면 바로 게임 시작
    if (num === 1) {
        setTimeout(() => {
            startGame();
        }, 500);
    } else {
        // 순서 결정 화면으로 이동
        showScreen('rpsScreen');
        autoRPSAndDetermineOrder();
    }
}
```

## 🎓 교육 효과

### 혼자 플레이 모드
- **장점**: 
  - 자신의 속도로 학습 가능
  - 실수해도 부담 없음
  - 반복 연습에 최적화
  - 집중적인 나눗셈 학습

### 2명 플레이 모드
- **장점**:
  - 경쟁을 통한 동기부여
  - 친구/가족과 함께 학습
  - 전략적 사고 훈련
  - 협력 및 소통 능력 향상
