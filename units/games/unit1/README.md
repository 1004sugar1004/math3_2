# 🎮 막상막하 곱셈 빙고

초등학교 3학년 수학 교육을 위한 재미있는 웹 기반 곱셈 연습 게임입니다. 두 자리 수 곱셈을 빙고 게임으로 즐겁게 학습할 수 있습니다.

![게임 스크린샷](https://via.placeholder.com/800x400/667eea/ffffff?text=막상막하+곱셈+빙고)

## 📖 프로젝트 소개

**막상막하 곱셈 빙고**는 초등학교 3학년 수학 교재에서 영감을 받아 만든 교육용 웹 게임입니다. 학생들이 곱셈을 재미있게 연습하면서 전략적 사고와 계산 능력을 동시에 키울 수 있습니다.

### 🎯 학습 목표
- 두 자리 수 × 두 자리 수 곱셈 연습
- 빠른 암산 능력 향상
- 전략적 사고 능력 배양
- 친구들과의 협력 및 경쟁 학습

## 🎲 게임 규칙

### 기본 플레이 방법

1. **순서 정하기**
   - 가위바위보로 순서를 정합니다 (웹에서는 자동으로 순서 설정)

2. **숫자 선택**
   - 첫 턴: 금화와 보석에서 각각 하나씩 자유롭게 선택
   - 두 수를 곱한 결과가 자동으로 계산됩니다

3. **빙고판 색칠**
   - 계산 결과를 빙고판에서 찾아 클릭
   - 해당 칸이 플레이어 색으로 색칠됩니다

4. **클립 이동 규칙** ⚠️ 중요!
   - 다음 플레이어는 **두 클립 중 하나만 같은 줄로** 이동할 수 있습니다
   - 금화를 움직이기로 했다면 → 금화 영역 내 다른 숫자로만 이동
   - 보석을 움직이기로 했다면 → 보석 영역 내 다른 숫자로만 이동
   - 예: 금화 43을 62로 바꾸고, 보석은 그대로 유지

5. **승리 조건**
   - 가로, 세로, 대각선 중 하나라도 **5개**를 먼저 색칠한 플레이어가 승리!

## 🎵 사운드 시스템 상세

### Web Audio API 활용

브라우저 내장 Web Audio API를 사용하여 **API 키나 외부 파일 없이** 순수 JavaScript로 모든 사운드를 생성합니다.

#### 사운드 종류 및 구현 방식

| 사운드 | 이벤트 | 구현 방식 | 특징 |
|--------|--------|-----------|------|
| 🪙 금화 선택 | 금화 버튼 클릭 | 다중 주파수 Sine 파형 (800-1500Hz) | 메탈릭한 동전 소리 |
| 💎 보석 선택 | 보석 버튼 클릭 | Triangle 파형 상승 음계 (523-1175Hz) | 반짝이는 마법 소리 |
| ✅ 정답 | 정답 빙고판 클릭 | Square 파형 도-미-솔 화음 | 성공 멜로디 |
| ❌ 오답 | 오답 빙고판 클릭 | Sawtooth 파형 하강 (200→100Hz) | 버저 소리 |
| 🎯 칸 색칠 | 빙고판 색칠 성공 | Sine 파형 하강 (400→200Hz) | 페인트 스와이프 |
| 🎊 빙고 | 빙고 완성 | 팬파레 멜로디 + 하모니 | 승리 축하 사운드 |
| 🔄 턴 전환 | 다음 턴 시작 | 딩-동 2음 (660Hz, 523Hz) | 부드러운 전환 |
| 🎮 게임 시작 | 게임 시작 버튼 | Triangle 파형 상승 음계 | 시작 알림 |

#### 기술적 특징

```javascript
// Oscillator: 파형 생성기
oscillator.type = 'sine' | 'square' | 'triangle' | 'sawtooth';
oscillator.frequency.value = 440; // Hz 단위

// GainNode: 볼륨 제어
gainNode.gain.setValueAtTime(0.3, time);
gainNode.gain.exponentialRampToValueAtTime(0.01, time + duration);

// 다중 레이어 사운드로 풍부한 음향 구현
```

#### 브라우저 호환성
- ✅ Chrome/Edge: 완벽 지원
- ✅ Firefox: 완벽 지원
- ✅ Safari: webkit 접두사로 지원
- ⚠️ 모바일: 첫 사용자 인터랙션 후 활성화 (정책)

### v1.1 - 화면 최적화 업데이트 (2025-11-07)

**PC와 태블릿에서 스크롤 없이 한 화면에 모든 요소 표시**

#### 주요 개선 사항

1. **레이아웃 압축**
   - 컨테이너 max-height: 97vh로 조정
   - 모든 간격(spacing) 변수 축소
   - 헤더, 턴 표시, 플레이어 현황 영역 압축

2. **요소 크기 최적화**
   - 금화/보석 버튼: 54px × 54px
   - 빙고판: 데스크톱 420px, 태블릿 380-400px
   - 빙고 셀: min-height 45px
   - 폰트 크기 전반적 축소

3. **반응형 개선**
   - 1400px 이상: 최대 크기 레이아웃
   - 1201-1400px: 중간 크기 최적화
   - 1200px 이하: 세로 레이아웃 전환
   - 각 브레이크포인트별 세밀한 조정

4. **시각적 개선 유지**
   - 압축하면서도 가독성 유지
   - 애니메이션 효과 보존
   - 버튼 디자인 품질 유지
