# 🍔 햄버거 만들기

초등학생을 위한 분수 학습 교육용 게임입니다. 손님의 주문을 받아 주문서를 작성하고, 햄버거를 만들면서 분수 개념을 재미있게 배울 수 있습니다.

## 📋 프로젝트 개요

이 프로젝트는 초등학생들이 분수의 개념을 실생활 예제(햄버거 만들기)를 통해 자연스럽게 학습할 수 있도록 설계된 인터랙티브 웹 애플리케이션입니다.

## ✨ 주요 기능

### 현재 완성된 기능

1. **시작 화면**
   - 게임 소개 및 손님의 주문 내용 표시
   - 3단계 소개 카드
   - 게임 시작 버튼
   - 3단계 바로가기 버튼

2. **1단계: 주문서 작성**
   - 손님의 주문 내용 표시 (달걀부침 1개, 고기 패티 1개, 치즈 3개, 양배추 4개)
   - 우리가 가진 재료 현황 표시 (시각적 카드 형태)
     - 달걀부침: 총 3개
     - 고기 패티: 총 4개
     - 치즈: 총 6개
     - 양배추: 총 8개
   - 주문서 작성 기능
     - 각 재료별로 분수 형태로 표현 (예: 1/3, 1/4, 3/6, 4/8)
     - 분자 입력 검증 기능
     - 정답 확인 및 피드백

3. **2단계: 햄버거 만들기**
   - 완성된 주문서 표시
   - 드래그 앤 드롭 방식의 재료 추가
   - 재료 팔레트 (달걀부침, 고기 패티, 치즈, 양배추, 빵)
   - 순서대로 재료 쌓기 기능
     - 다음 재료 안내 표시
     - 올바른 순서와 개수 검증
   - 재료 제거 기능
   - 햄버거 완성 확인
   - 시각적 피드백 (애니메이션 효과)

4. **인터랙션 기능**
   - 부드러운 화면 전환 애니메이션
   - 드래그 앤 드롭 + 클릭 상호작용
   - 성공/오류 모달 팝업
   - 호버 효과 및 시각적 피드백
   - 키보드 단축키 지원 (ESC로 모달 닫기)
   - 단계 간 이동 버튼 (처음으로, 3단계로 가기)

5. **3단계: 도전 모드**
   - 랜덤 주문서 생성 기능
   - 분수 형태로 주문 제시 (예: "양배추 8장의 3/8")
   - 자유로운 순서로 재료 쌓기
   - 정답 확인 및 피드백
   - 새로운 주문서 생성 기능
   - 무한 반복 학습 가능

6. **인터랙션 기능**
   - 부드러운 화면 전환 애니메이션
   - 드래그 앤 드롭 + 클릭 상호작용
   - 성공/오류 모달 팝업
   - 호버 효과 및 시각적 피드백
   - 키보드 단축키 지원 (ESC로 모달 닫기)

7. **반응형 디자인**
   - 데스크톱, 태블릿, 모바일 화면 대응
   - 터치 인터페이스 지원

## 🎮 게임 구성

이 게임은 **3단계**로 구성되어 있으며, 단계별로 난이도가 높아집니다.

## 🎯 게임 진행 방식

### 1단계: 주문서 작성 📝
1. 손님의 주문 내용을 확인합니다
   - "달걀부침 1개, 고기 패티 2개, 치즈 3개, 양배추 2개"
2. 우리가 가진 재료의 총 개수를 확인합니다
   - 달걀부침: 3개 / 고기 패티: 4개 / 치즈: 6개 / 양배추: 8개
3. 주문서 표에 각 재료의 주문 수량을 **분수로 표현**합니다
   - 예: 달걀부침 1개 = **1/3** (전체 3개 중 1개)
   - 예: 고기 패티 2개 = **2/4** = **1/2** (전체 4개 중 2개)
   - 예: 치즈 3개 = **3/6** = **1/2** (전체 6개 중 3개)
   - 예: 양배추 2개 = **2/8** = **1/4** (전체 8개 중 2개)
4. **"주문서 완성!"** 버튼을 클릭합니다
5. 모든 분수를 올바르게 입력하면 **자동으로 2단계로 이동**합니다!

### 2단계: 햄버거 만들기 🍔
1. 접시 위에 **아래 빵**이 이미 깔려있습니다 🍞
2. 왼쪽에 표시된 **완성된 주문서**를 확인합니다
   - 전체 수량과 분수 표현을 먼저 확인
   - 주문량은 **네모 박스**로 표시 (처음엔 0개)
3. 화면 중앙에 표시되는 **"다음 재료"**를 확인합니다
4. 오른쪽 재료 팔레트에서 해당 재료를 **클릭하거나 드래그**합니다
   - 클릭: 재료를 클릭하면 즉시 햄버거에 추가됩니다
   - 드래그: 재료를 드래그하여 중앙 접시에 드롭합니다
5. **반드시 달걀부침부터 순서대로** 재료를 쌓아야 합니다
   - 순서: 🍞 아래 빵 → 🍳 달걀부침(1개) → 🥩 고기 패티(2개) → 🧀 치즈(3개) → 🥬 양배추(2개) → 🍔 위 빵
6. 재료를 쌓을 때마다 **네모 박스의 숫자가 증가**합니다
   - 🟢 정답: 초록색 박스
   - 🟡 진행중: 노란색 박스  
   - 🔴 오답: 빨간색 박스
7. 각 재료의 개수를 정확히 맞춰야 합니다
8. 마지막으로 **위 빵**을 올리면 **완성된 햄버거**로 변신합니다! 🍔
9. **"햄버거 완성!"** 버튼을 클릭하여 최종 확인합니다
10. 완벽하게 만들면 성공 메시지가 나타납니다! 🎉
11. **"다음 단계로"** 버튼을 클릭하여 3단계로 이동합니다

### 3단계: 도전! 햄버거 만들기 🎯
1. 컴퓨터가 **랜덤으로 생성한 주문서**를 확인합니다
2. 주문서에는 **재료 그림**과 **세로 분수**가 함께 표시됩니다
   - 재료 전체 개수를 작은 그림으로 확인
   - 예: "달걀부침 3장의 ²⁄₃" = 2개
   - 예: "치즈 6장의 ⁵⁄₆" = 5개
3. **분수를 계산**하여 필요한 재료 개수를 파악합니다
4. 재료를 클릭하거나 드래그하여 햄버거를 만듭니다
5. **순서는 자유롭게** 쌓을 수 있습니다
6. 마지막으로 위 빵을 올리고 **"햄버거 완성!"** 버튼을 클릭합니다
7. 정답이면 성공 메시지가 나타납니다! 🎉
8. **"새로운 주문서"** 버튼으로 계속 도전할 수 있습니다
